# Типи та характеристики SSD-накопичувачів

---

## Вступ

SSD (Solid State Drive) — це твердотільний накопичувач даних, який використовує флеш-пам’ять для зберігання інформації.  
На відміну від HDD, SSD не має рухомих частин, що забезпечує високу швидкість роботи, надійність і безшумність.

---

# 1. Типи SSD-накопичувачів

---

## 1.1 SATA SSD (2.5")

![SATA SSD 2.5](https://api.e-server.com.ua/storage/175308/rs/U0661741_big___rs_1200_1200.jpg)

**Характеристики:**
- Інтерфейс: SATA III
- Форм-фактор: 2.5 дюйма
- Максимальна швидкість читання: до 550 МБ/с
- Максимальна швидкість запису: до 520 МБ/с
- Найпоширеніший тип SSD

**Переваги:**
- Доступна ціна
- Сумісність із більшістю ПК та ноутбуків

---

## 1.2 M.2 SATA SSD

![M.2 SATA SSD](https://image-us.samsung.com/SamsungUS/home/computing/memory-and-storage/solid-state-drives/pdp/100318/gallery/MZ-N6E1T0BW_001_Front_Blackv2.jpg?$product-details-jpg$?$product-details-jpg$)

**Характеристики:**
- Форм-фактор: M.2
- Інтерфейс: SATA
- Швидкість читання: до 550 МБ/с
- Компактний розмір

**Переваги:**
- Відсутність кабелів
- Зручність встановлення

---

## 1.3 M.2 NVMe SSD (PCIe)

![M.2 NVMe SSD](https://dmassets.micron.com/is/image/microntechnology/product-2400-m2-group:3-2-all-others?ts=1733767550197&dpr=off)

**Характеристики:**
- Інтерфейс: PCI Express
- Протокол: NVMe
- Швидкість читання: 1500–7000 МБ/с
- Швидкість запису: 1000–6500 МБ/с

**Переваги:**
- Дуже висока продуктивність
- Підходить для ігор, обробки відео та професійної роботи

---

## 1.4 PCIe SSD (розширювальна карта)

![PCIe SSD](https://img.kwcdn.com/product/1e133b3590a/624935a1-99bb-495f-8cfa-a93b299eabce_1121x1121.jpeg?imageMogr2/auto-orient%7CimageView2/2/w/800/q/70/format/webp)
**Характеристики:**
- Встановлюється у слот PCI Express
- Надвисока швидкість передачі даних
- Використовується у серверних та професійних системах

---

## 1.5 Зовнішній SSD

![External SSD](https://i.citrus.world/imgcache/size_800/uploads/shop/8/d/8d8c1f0cd0ed89f03b57351df31f0f57.jpg)

**Характеристики:**
- Підключення через USB 3.2 або Type-C
- Портативність
- Використовується для резервного копіювання та перенесення даних

# 3. Скрипти для відображення технічної інформації про встановлений накопичувач

## 3.1 Windows (PowerShell)

```powershell
Get-PhysicalDisk | Format-Table FriendlyName, MediaType, Size, SerialNumber
```

---

## 3.2 Windows (CMD)

```cmd
wmic diskdrive get model,size,serialnumber
```

---

## 3.3 Linux (Terminal)

```bash
lsblk -o NAME,SIZE,TYPE,MODEL
```

або

```bash
sudo hdparm -I /dev/sda
```

---

## 3.4 Python-програма для Windows

```python
import wmi

c = wmi.WMI()

for disk in c.Win32_DiskDrive():
    print("Model:", disk.Model)
    print("Size:", round(int(disk.Size)/1024**3, 2), "GB")
    print("Serial Number:", disk.SerialNumber)
    print("-" * 40)
```

# Висновок

SSD-накопичувачі значно перевершують традиційні HDD за швидкістю, надійністю та енергоефективністю.  
Найсучаснішими та найпродуктивнішими є NVMe SSD, які використовують інтерфейс PCIe.  
Вибір типу SSD залежить від потреб користувача, бюджету та можливостей материнської плати.
